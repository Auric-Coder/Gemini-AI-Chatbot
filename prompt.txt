Display user messages on the right and AI responses on the left. Add a typing indicator while the AI is generating a response, automatically scroll to the latest message, and clear the input after sending.

Create a backend API endpoint like `/api/chat` that receives a user's message, sends it to the Gemini API using the official `@google/genai` SDK, and returns the generated response to the frontend. Include proper error handling for API failures or invalid requests.
For the frontend, create reusable React components instead of putting everything into one file. The UI should be responsive and look good on desktop and mobile. Use Tailwind CSS for styling and keep the design simple but modern.
Add a dark/light mode toggle, a "New Chat" button to clear the conversation, and show timestamps for each message. If possible, render Markdown responses and nicely formatted code blocks.
Also save the chat history in localStorage so the conversation is still available after refreshing the page.
At the end, generate a good README explaining how to install dependencies, where to add the Gemini API key, how to run both the frontend and backend locally, how to push the project to GitHub, and how to deploy the frontend on Vercel and the backend on Render.
